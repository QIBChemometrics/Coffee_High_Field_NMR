# Coffee_NMR_Spectra
600 MHz and 60 MHz NMR spectra of extracts of various ground roast coffees of known origin.

Lipophilic extract spectra are described in Gunning et. al. https://doi.org/10.1016/j.foodchem.2017.12.034
